## ï¿½ Key Features

### 1. **Aurora & Glassmorphism UI**
A custom-built `AuroraBackground` component using CSS gradients and blurs to create a living, breathing backdrop that reacts subtly to the user's presence.

### 2. **Bento Grid Architecture**
The **Expertise Section** utilizes a responsive "Bento Grid" layout (`BentoItem.js`), mimicking modern dashboard aesthetics to present diverse skills (Leadership, Dev, Analytics) in a unified, visually rich format.

### 3. **Cinematic Rows**
The **Projects & Certifications** sections feature `CinematicRow.js`â€”a component that transforms list items into interactive cinematic strips. Hovering over a project reveals hidden details, action buttons, and lighting effects.

### 4. **Magnetic Interactions**
Custom hooks implement magnetic physics on buttons (`MagneticButton.js`), causing actionable elements to "stick" to and follow the cursor, enhancing the feeling of fluidity.

## ðŸš€ Local Development

To run this project locally, follow these steps:

1.  **Clone the repository**
    ```bash
    git clone https://github.com/hatim3310/portfolio-v2.git
    cd portfolio-v2
    ```

2.  **Install Dependencies**
    ```bash
    npm install
    ```

3.  **Start Dev Server**
    ```bash
    npm start
    ```
    Open [http://localhost:3000](http://localhost:3000) to view it in the browser.
